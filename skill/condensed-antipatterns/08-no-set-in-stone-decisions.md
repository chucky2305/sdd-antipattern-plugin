## AP-08: Not mentioning decisions already made

**What it means:** When locked-in decisions aren't written down, people waste time evaluating options that were never on the table, and an AI agent quietly picks its own.

**What a good spec includes:**
- A "Decisions already made" section with: tech stack choices, vendors, architectural patterns, anything locked in and not up for debate
- Brief reason for each locked decision (budget, existing infra, compliance, etc.)
